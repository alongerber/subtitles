# אולפן הקול — ElevenLabs Studio

קול → תזמון מדויק לכל מילה → עיצוב כתוביות חי. הכל בכלי אחד, רץ מדומיין בלי CORS.

## הפעלה (פעם אחת, ואז דומיין קבוע)

**אופציה A — Vercel CLI:**
```
cd el-studio
npx vercel        # התחברות + פריסה. נותן כתובת https קבועה.
```

**אופציה B — GitHub → Vercel:**
1. דחוף את התיקייה לריפו ב-GitHub.
2. ב-vercel.com → Add New → Project → Import את הריפו → Deploy.

זהו. אין build, אין הגדרות. `index.html` מוגש כסטטי, ו-`/api/*` הן פונקציות serverless.

## איך זה עוקף CORS
`api/el.js` מעביר את כל הקריאות ל-ElevenLabs בצד השרת. הדפדפן מדבר רק עם הדומיין שלך.
המפתח (xi-api-key) נשלח רק לדפלוימנט שלך — לא לשום צד שלישי. נשמר ב-localStorage של הדפדפן.

`api/nakdan.js` עושה אותו דבר מול ה-Nakdan של דיקטה (ניקוד עברית).

## הערות
- אם פותחים את `index.html` ישירות (file://) זה ינסה לקרוא ישירות ל-ElevenLabs — יעבוד רק אם CORS מאושר. הדרך הבטוחה היא הדומיין.
- תזמון ברמת מילה: מובטח ב-multilingual_v2 / flash / turbo. ב-v3 בדרך כלל מוחזר alignment; אם לא — הכלי מתריע.
- ה-JSON שמיוצא ממודול העיצוב (`caption-style.json`) כולל סגנון + תזמון + טקסט — מוסרים אותו לשריפה סופית מדויקת לפיקסל.
